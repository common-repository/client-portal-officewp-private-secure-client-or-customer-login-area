== Description ==

<h3 style="margin-bottom:20px;"><a href="https://officewp.com">OfficeWP Starter :: Private Client Portals</a></h3>
<p>Visit Website &gt;&gt; <a href="https://officewp.com">OfficeWP.com</a> is the home of OfficeWP Pro, a premium, fully featured Client Portal plugin for WordPress.</p>
<h3 style="margin-bottom:20px;">IMPORTANT :: What is OfficeWP Starter?</h3>
<p>OfficeWP Starter is designed to give just an introduction to the very basic components of a Client Portal, but there are many features in the PRO version that are not included in the Starter version.</p>
<p>OfficeWP Starter functions as a very basic Client Portal, but at the same time, provides a glimpse into what is included and possible in the OfficeWP PRO version.</p>
<p>OfficeWP Pro is in constant development and is fully supported. New features are added regularly, and your suggestions are welcome.</p>
<p>See a full list of features at <a href="https://officewp.com">OfficeWP.com</a></p>
<h3>Brief Description</h3>
<p>Easily configure multiple areas of your WordPress installation and allows the site Administrator to quickly create new Client Areas, Client Management Portals, Client File Upload Areas, or Private Staff Pages on the site by entering just a few data fields and clicking submit. Additionally, clients can upload or download secure files.</p>


== Installation ==

This section describes how to install the Starter version of the plugin and get it working.
<ul>
<li>Upload the newest version using 'Add new' Plugin (WordPress -> Plugins -> Add New), Upload via zip, Upload, and then Activate - OR - Unzip the files on your local machine and upload via FTP to the /plugins/ directory.. then activate from the plugins list.</li>
</ul>
<h3 style="margin-bottom:10px;">OfficeWP Starter :: Private Client Portals :: Installing & Activating OfficeWP PRO</h3>
<ul>
<li>When you upgrade to PRO, you will need first to completely delete the OfficeWP Starter files from your installation. Your data, however, as far as clients, files, etc... will all be safely stored in the database & proprietary upload directory.</li>
</ul>
